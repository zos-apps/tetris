## About

The legendary tile-matching puzzle game.

## Features

- Classic gameplay
- Ghost piece
- Hold piece
- Score tracking
- Levels

## Installation

### From App Store
Open the **App Store** in zOS and search for "Tetris".

### Via npm
```bash
npm install @zos-apps/tetris
```

## Usage

- Arrow keys to move
- Up to rotate
- Space for hard drop
